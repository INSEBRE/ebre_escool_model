Laravel model for old ebre_escool database

## Install

Via Composer

``` bash
$ composer require scool/ebre_escool_model
```

## Usage

``` php
$skeleton = new Scool\EbreEscoolModel();
echo $skeleton->echoPhrase('Hello, League!');
```
